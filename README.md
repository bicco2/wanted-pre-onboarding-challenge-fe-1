# 원티드 프리온보딩 챌린지 프론트엔드 코스 사전과제

1. 로그인 회원가입 기능 구현

   1. 로그인시 응답으로 받은 토큰을 로컬 스토리지에 저장 후 토큰이 존재하면 리다이렉트 만료면 다시 로그인
   2. 로그인은 조건이 만족할 때 제출버튼 활성화
   3. 조건 : 이메일(@ , . 포함), 비밀번호 (8자리 이상)

2. 투두리스트 기능 구현
